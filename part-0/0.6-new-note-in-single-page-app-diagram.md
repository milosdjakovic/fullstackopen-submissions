# 0.6: New Note in Single Page App Diagram

```mermaid
sequenceDiagram
    participant browser
    participant server

    Note right of browser: User writes a new note and clicks the Save button

    Note right of browser: JavaScript pushes the new note into local notes array and redraws the page

    browser->>server: POST https://studies.cs.helsinki.fi/exampleapp/new_note_spa
    activate server

    Note left of server: Server saves the note and responds with {"message":"note created"}

    server-->>browser: HTTP 201 Created
    deactivate server
```
